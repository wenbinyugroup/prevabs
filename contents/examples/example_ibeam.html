I-beam#

This example has an I-shape cross section. The dimensions shown in Fig. %s are \(w_1=2.0\) m, \(w_2=3.0\) m, \(h=3.0\) m, \(t_1=0.11\) m, \(t_2=0.065\) m, \(t_w=0.08\) m. Materials and layups are given in Table %s and Table %s. The effective stiffness matrix is listed in Table %s. Complete input files can be found in examples\ex_ibeam\, including ibeam.xml, basepoints.dat, baselines.xml, materials.xml, and layups.xml.
