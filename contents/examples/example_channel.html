Channel#

Problem description#

This example has a cross section of a highly heterogeneous channel. This cross section geometry can be defined as shown in Fig. 36 [CHEN2010]. The isotropic material properties are given in Table 18. The layup is defined having a single layer with the thickness 0.001524 m. The result is shown in Table 20 and compared with those in [CHEN2010]. Complete input files can be found in examples\ex_channel\, including channel.xml and materials.xml.
